Printing Subscription? What You Need to Know

Yes, Odoo Direct Print PRO requires a printing subscription to operate - and we believe in being fully transparent about it.

  • Cloud-Based Print Engine
    Your print jobs are securely routed from Odoo to your computers through our cloud print server - no extra hardware, no complex networking.
  • Minimal Load on Odoo
    The only thing your Odoo does is generate a report and send it to our print API. No additional calls, loops, or traffic inside your Odoo server.
  • Continuous Improvements

Flexible Printing Scenarios

Automate your printing with 20+ built-in triggers. Each scenario is built from a powerful print action (like “Print after transfer validation”) combined with optional filters - giving you full control over what, when, and how documents are printed.

  • → Print document on Transfer (after validation)
  • → Print document on Backorder
  • → Print Packages Label on Transfer
  • → Print Picking Document after Sales Order Confirmation
  • → Print document on Purchase Order
  • → Print product labels on Transfer
  • → Print lot labels on Transfer
  • → Print Invoice Document after validation
  • → Print Package on Put in Pack
  • → Print Document on Picking Status Change

You can extend any scenario with conditions - for example, only print if a sales order comes from Shopify, or skip documents for international shipments. Looking for something custom? Build your own logic with Print Action Buttons or custom rules.
